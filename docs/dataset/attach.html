attach

Syntax

    dataset attach JSON_RECORD_ID FILENAME(S)

Description

Attach a file to a JSON record. Attachments are stored in a tar ball related to the JSON record id.

Usage

Attaching a file named start.xlsx to the JSON record with id t1

    dataset attach t1 start.xlsx

Related topics: attachments, detach and prune
