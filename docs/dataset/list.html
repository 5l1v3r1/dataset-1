list

Syntax

    dataset list JSON_RECORD_IDS

Description

The writes a JSON array to standard out (unless you’ve specific an alternative location with the “-output” option) for the provided JSON_RECORD_IDS for each ID found. If no ids are provided (or none are found) then an empty JSON array is return.

Usage

An example we’re assuming there is a JSON document with a JSON_RECORD_ID of “r1”.

    dataset list r1

If “r1” was ‘{“one”:1}’ then the output would be

    [{"one":1}]
