read

Syntax

    dataset read JSON_RECORD_ID

Description

The writes the JSON document to standard out (unless you’ve specific an alternative location with the “-output” option) for the given JSON_RECORD_ID.

Usage

An example we’re assuming there is a JSON document with a JSON_RECORD_ID of “r1”.

    dataset read r1
