Configuration files

For each system that is aggregated you need to have a configuration file. The scripts for building feeds expects

  • etc/CaltechAUTHORS.bash
  • etc/CaltechTHESIS.bash
  • etc/CaltechARCHIVES.bash
  • etc/CaltechPEOPLE.bash
  • etc/orcid-api.bash

The first two can be templated from etc/eprints.bash-example and the later from etc/archivesspace.bash-example. For details on harvesting from EPrints look at documentation in https://github.com/caltechlibrary/epgo and for details for harvesting from ArchivesSpace look at https://github.com/caltechlibrary/cait.
