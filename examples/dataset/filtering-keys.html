Filters and sorts

dataset keys can be used to to filter and sort a key list. Here’s is a simple case for match records where given name is equal to “Mojo”. We will save the result in a file called mojo.keys.

   dataset keys '(eq .given "Mojo")' > mojo.keys

You can also use an existing key list (e.g. mojo.keys) to sub select based on a new filter and/or sort expression. We will filter for a family name of “Sam” and sort by the age field.

   dataset -key-file=mojo.keys keys '(eq .family "Sam")' '+.age'
