Description

dsfind is a command line tool for querying a Bleve indexes based on the records in a dataset collection. By default dsfind is assumed there is an index named after the collection. An option lets you choose different indexes to query. Results are written to standard out and are paged. The query syntax supported is described at http://www.blevesearch.com/docs/Query-String-Query/.

Options can be used to modify the type of indexes queried as well as how results are output.
